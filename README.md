# Financial Analysis Dashboard

## Project Overview
This project is a Financial Analysis Dashboard built using Power BI to analyze sales, profit, and financial performance across different countries and months.

## Features
- Sales Analysis
- Profit Analysis
- Country-wise Insights
- Month-wise Trends
- Interactive Dashboard
- Financial Performance Tracking

## Tools Used
- Power BI
- Excel
- Data Visualization
- DAX

## Dashboard Visuals
- Gauge Chart
- Donut Chart
- Area Chart
- Bar Chart
- Stacked Area Chart

## Purpose
The purpose of this project is to generate meaningful business insights using interactive visualizations.
